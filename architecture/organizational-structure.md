## 三、决策流程

**路径一（梦想家发起）：**
梦想家提出问题/任务 → 阿尔法与各有关部门协商出方案 → 请示梦想家 → 梦想家决策 → 阿尔法执行 → 监察 → 汇报

**路径二（方向负责人发起）：**
方向负责人提出 → 阿尔法审批 → 执行 → 监察 → 汇报

**重大决策：**
发现问题 → 阿尔法与各有关部门协商出方案 → 请示梦想家 → 梦想家决策 → 阿尔法执行 → 汇报结果

**紧急决策：**
风险发现 → 监察安全部立即上报 → 阿尔法评估 → 直接通知梦想家 → 先处置后补流程

## 四、核心制度（待研发部细化）

4. **协作规范** — 负责人统筹，协作人配合，跨部门约塔协调
