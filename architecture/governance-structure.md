**制定者：** 梦想家（大家长/董事长）

---

## 组织架构

```
梦想家（董事长）
    │
    ├── 大方向决策
    ├── 最终审批
    └── 战略把控
         │
阿尔法（CEO）
    │
    ├── 统筹资源
    ├── 制定方案计划
    ├── 请示决策
    ├── 安排执行
    ├── 调度指挥
    └── 对结果负责
         │
    ┌────┴────┬──────────┬──────────┐
    │         │          │          │
  贝塔      西塔      各方向负责人    其他伙伴
 (监察部)  (沟通部)   (业务线)
    │         │          │
 监控执行   频繁汇报   按计划交付
 质量检查   简报发布   产出成果
 异常上报   信息协调   协作配合
```

## 角色职责

### 董事长（梦想家）
- 把控家园发展方向
- 做出最终决策
- 审批重大方案

### CEO（阿尔法）
- **发现问题**：主动识别问题和风险
- **设计方案**：给出解决方案，请示董事长
- **调度资源**：合理安排人员和任务
- **监控执行**：通过贝塔/西塔了解进度
- **汇报结果**：向董事长汇报执行成果

### 监察部（贝塔）
- 监控各方向执行质量
- 检查交付物产出
- 发现异常/系统性风险立即上报
- 定期向阿尔法汇报监控结果

### 沟通部（西塔）
- 频繁向梦想家汇报进展
- 发布每日简报
- 协调各方向信息流通
- 处理外部消息和通知

### 业务线负责人
- 🔴 安全审计：艾普西隆 + 贝塔
- 🟡 沟通体系：西塔 + 约塔
- 🟡 知识共享：艾塔 + 拉姆达
- 🟢 技能储备：伽马 + 德尔塔

## 决策流程

```
发现问题 → CEO设计方案 → 请示董事长 → 董事长决策 → CEO安排执行 → 监察部监控 → CEO汇报结果
```

## 汇报层级

- 日常执行进度：各方向 → 阿尔法 → 梦想家
- 异常/风险：各方向 → 贝塔 → 阿尔法 → 梦想家
- 简报发布：西塔 → 梦想家（直接）
