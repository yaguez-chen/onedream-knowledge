## 风险控制

| 风险 | 应对 |
|------|------|
| Exec白名单配置导致Agent无法工作 | 回滚：恢复备份 + 重启网关（<5分钟） |
| 网关重启后部分Agent未恢复 | 贝塔逐个检查，手动重启异常Agent |
| 白名单过宽失去安全意义 | 贝塔审查白名单，宁可过严不可过松 |
| 配置错误导致网关无法启动 | 恢复备份配置，不需要改其他东西 |
